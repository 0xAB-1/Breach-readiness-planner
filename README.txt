Breach Readiness Planner

Description:
This project is a Python-based web application that generates a
customized breach readiness and incident response plan for small
businesses based on user inputs.

Technology Used:
- Python
- Flask
- HTML / Bootstrap

How to Run the Application:
1. Install Python 3.x
2. Open a terminal in the project folder
3. Run: python app.py
4. Open a browser and go to: http://127.0.0.1:5000

Project Features:
- Business data inventory form
- Breach scenario selection
- Risk scoring and summary
- First 24-hour incident response checklist
- Printable PDF-style output

Note:
This application is for educational purposes and provides
preparedness guidance only.
